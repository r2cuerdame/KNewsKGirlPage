Privacy Policy

Last updated: January 30, 2026

Overview

KNewsKGirl is a personal automation project that uploads videos to YouTube. This application is used solely by the developer for managing a personal YouTube channel.

Data Collection

This application:

  • Does NOT collect personal data from users
  • Does NOT track visitors or viewers
  • Does NOT share any data with third parties

YouTube API Usage

This application uses the YouTube Data API v3 to:

  • Upload videos to a personal YouTube channel
  • Manage video metadata (titles, descriptions, captions)
  • Read video performance analytics

All YouTube API usage complies with YouTube Terms of Service and Google Privacy Policy.

Data Storage

  • OAuth tokens are stored securely in GitHub Secrets
  • No user data is collected or stored
  • Video content is stored only on YouTube's servers

Third-Party Services

This application integrates with:

  • YouTube Data API v3 - For video uploads
  • OpenAI API - For content generation
  • ElevenLabs API - For text-to-speech
  • Hedra API - For video generation

Each service has its own privacy policy.

Contact

For questions about this privacy policy, please open an issue on the GitHub repository.

Changes

This privacy policy may be updated occasionally. Changes will be reflected in the "Last updated" date.
